## Local development
### Prerequisites
- Node.js v18.19.x, could work on older or newer versions
- Yarn package manager v1.x

### Set up
Install the dependencies
```
yarn
```

Run the HTTP server
```
yarn serve
```

Run the test suite
```
yarn test
```

For more, refer to the scripts defined in `package.json`.
